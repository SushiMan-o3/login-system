# Overview
A basic login system implemented by Sushim Malla to figure out how react.js and FastAPI can work together to build a login system.

Made in order to be implemented for other future projects in mind. 

# Project Setup Guide

## Clone the Repository

Nagivate to the desired folder where you want to clone the repoistory then clone the repository.

```bash
git clone https://github.com/SushiMan-o3/login-system.git
```

Navigate to the repository. 
```bash
cd <login-system>
```

## Backend Setup

1. Navigate to the backend directory.

```bash
cd backend
```

2. Install the required Python dependencies.

```bash
pip install -r requirements.txt
```

3. Run the backend server.

```bash
python main.py
```

---

## Frontend Setup

1. Ensure that Node.js and npm are installed on your system.

2. Navigate to the frontend directory. 

```bash
cd frontend
```

3. Install all project dependencies. 

```bash
npm install
```

4. Start the development server. 

```bash
npm start
```

5. After running the command above, a **local development URL** will appear in the terminal.
   Click the generated link to open the application in your browser.
